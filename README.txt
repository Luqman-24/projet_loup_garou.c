Notre Loup-Garou jouable sur différents OS.
Tout d'abord, télécharger le fichier compressé correspondant à votre système d'exploitation (Linux ou Windows).
Les principales différences entre les deux versions sont ; les commandes qui permet de supprimer le terminal (clear et cls) et
les bibliothèques externes qui sont utilisées différemment.
--------------------------------------------------LINUX-------------------------------------------------

Le programme nécessite SDL2 et SDL2_mixer (pour le son)
Décompresser le fichier.
Rendez-vous dans le terminal, puis, dans le dossier bin du jeu, exécuter "./projet"
Le jeu se démarrera directement. Une page nommée "Sons" s'ouvre, réduisez-la, elle sert uniquement à la musique et voix du narrateur.
Notre jeu se joue de façon très simple. Indiquer le nombre de joueurs puis suivez les instructions du guide. Celui-ci va demander
à chacun des joueurs de rentrer son pseudonyme et de prendre connaissance de son rôle.
Après ça, le narrateur est juste présent pour réveiller les joueurs.
Attention, l'entièreté du jeu se joue avec des nombres.
Amusez-vous bien :)

---------------------------------------------------------------------------------------------------------

--------------------------------------------------WINDOWS-------------------------------------------------

SDL2 et SDL2_mixer sont intégrés dans le fichier téléchargé.
Décompresser le fichier.
Rendez-vous dans le dossier /bin et exécuter le projet.exe. Certaines fois, Windows Defender vous empêche de lancer le jeu, sélectionner
alors, "Exécuter quand même".
Le jeu se démarrera directement. Une page nommée "Sons" s'ouvre directement en arrière-plan. Suivez les instructions sur le terminal.
Notre jeu se joue de façon très simple. Indiquer le nombre de joueurs puis, suivez les instructions du guide. Celui-ci va demander
à chacun des joueurs de rentrer son pseudonyme et de prendre connaissance de son rôle.
Après ça, le narrateur est juste présent pour réveiller les joueurs.
Attention, l'entièreté du jeu se joue avec des nombres.
Amusez-vous bien :)
